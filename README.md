# 🤖 Meena AI Chatbot

An advanced full stack AI assistant platform built using React, Flask, MongoDB Atlas, and Gemini AI.
The application supports intelligent conversations, PDF-based question answering, voice input, authentication, and cloud deployment.

---

# 🚀 Live Demo

Frontend: https://meena-ai-chatbot.vercel.app
Backend: https://meena-ai-chatbot.onrender.com

---

# 📌 Features

* 🤖 AI Chat using Gemini API
* 🔐 JWT Authentication System
* 🔒 Secure Password Hashing with bcrypt
* 💬 Multi Chat Support
* 🧠 Persistent Chat Storage
* 📄 PDF Question Answering
* 🎤 Voice Input Support
* 🌙 Dark / Light Mode
* ☁️ MongoDB Atlas Cloud Database
* 📱 Responsive UI
* 📝 Markdown Response Rendering
* 🚀 Cloud Deployment with Render & Vercel

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/santhiya-cloud2185/meena-ai-chatbot.git
```

---

## Frontend Setup

```bash
cd ai-chatbot-frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
pip install -r requirements.txt
python server.py
```

---

# 🔑 Environment Variables

Create a `.env` file in the backend folder.

```env
GEMINI_API_KEY=your_gemini_api_key

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```
